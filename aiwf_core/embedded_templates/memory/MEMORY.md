# Memory Index

Planner scans this before planning. Each line is a one-sentence hook.
Open a note only when the current work touches that topic.

Keep entries short. This file is an index, not a memory. Put details in
`notes/<topic>.md`.

Use notes only for stable guidance Planner would otherwise rediscover across
tasks. Do not index temporary progress, task status, guesses, or one-task-only
details.
