# Deferred Findings

Use this note only when an unresolved finding must survive across Tasks and no
downstream Task owns it yet. If the owner is known, write it into that Task
instead.

For each open item, keep only its source Task/observation, what remains, and the
condition that makes it relevant. Remove it when it moves into a Task, is
resolved, is accepted as a limitation, or is dismissed. The source Task record
keeps the history.
